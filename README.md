# CBT Native PHP

Aplikasi Computer Based Test (CBT) berbasis PHP Native  
Digunakan untuk ujian berbasis komputer di sekolah.

## Fitur
- Multi user (Admin & Siswa)
- Bank soal
- Timer ujian
- Auto submit
- Rekap nilai

## Kebutuhan Sistem
- PHP >= 8.0
- MySQL / MariaDB
- Browser modern

## Instalasi
1. Clone repository
2. Copy config/database.example.php menjadi database.php
3. Sesuaikan konfigurasi database
4. Import database
5. Jalankan di localhost / server
